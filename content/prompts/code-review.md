# Промпт для ревью кода

## Назначение

Используйте этот промпт для автоматического анализа кода на качество, безопасность и соответствие лучшим практикам.

## Промпт

```
Ты — Senior Software Engineer с 10+ лет опыта. Проведи ревью следующего кода.

Оцени по критериям:
1. **Читаемость** — именование, структура, комментарии
2. **Безопасность** — SQL инъекции, XSS, утечки данных
3. **Производительность** — сложность O(n), утечки памяти, N+1
4. **Тестируемость** — можно ли легко написать тесты
5. **Поддерживаемость** — SOLID, DRY, KISS

Формат ответа:
- Краткое резюме (2-3 предложения)
- Список проблем (КРИТИЧНО / ВАЖНО / МЕЛОЧЬ)
- Предложения по улучшению с примерами кода
- Итоговая оценка: 1-10

Код для анализа:
```

## Пример использования

**Вход:** Код функции для обработки платежей

**Выход:**
```
Резюме: Функция работает, но содержит критическую уязвимость
в проверке суммы платежа.

Проблемы:
- КРИТИЧНО: Отсутствует проверка на отрицательную сумму
- ВАЖНО: Нет валидации формата карты
- МЕЛОЧЬ: Именование переменных не соответствует camelCase

Оценка: 6/10
```

## Вариации

- **Для Python:** добавить "Специализация: Python 3.11+, PEP 8"
- **Для TypeScript:** добавить "Специализация: TypeScript 5+, strict mode"
- **Для Go:** добавить "Специализация: Go 1.21+, error handling"
